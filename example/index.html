Progress Circle Examples

Values

Like any progress bar, the circle can display values from 0 to 100. This example uses javascript to update the data-progress attribute and related .text element content only. The rest of the display is handled by the CSS.

0%

The example above uses the following LESS code:

    .progress-circle-value-example-1
    {
      .progressCircleColor(transparent, transparent, @blue, @orange);

      .text
      {
        .border-radius(50%);

        background-color: @white;
        border: 5px solid @yellow;
        font-weight: bold;
        margin: 4px;
      }

      &.progress-circle-large
      {
        .progressCircleSize(3, (5px + 4px));

        .text
        {
          font-size: (@baseFontSize * 2);
        }
      }
    }

The CSS output would look like the following:

    .progress-circle-value-example-1 .progress-circle-bg {
      background-color: transparent;
      border-color: transparent;
    }
    .progress-circle-value-example-1 .bar {
      border-color: #049cdb;
    }
    .progress-circle-value-example-1 .text {
      color: #f89406;
    }
    .progress-circle-value-example-1 .text {
      -webkit-border-radius: 50%;
      -moz-border-radius: 50%;
      border-radius: 50%;
      background-color: #ffffff;
      border: 5px solid #ffc40d;
      font-weight: bold;
      margin: 4px;
    }
    .progress-circle-value-example-1.progress-circle-large {
      width: 150px;
      height: 150px;
    }
    .progress-circle-value-example-1.progress-circle-large .progress-circle-bg,
    .progress-circle-value-example-1.progress-circle-large .bar {
      border-width: 15px;
    }
    .progress-circle-value-example-1.progress-circle-large .text {
      padding: 21px 0;
      font-size: 42px;
      line-height: 60px;
    }
    .progress-circle-value-example-1.progress-circle-large .text {
      font-size: 28px;
    }

The examples above use the following LESS code:

    .progress-circle-color-example-1
    {
      .progressCircleColor(@blue, @red, @green, @yellow);

Text Size

With the LESS mixin .progressCircleSize(), you can adjust the circle size if you needed to place a bit of text that displays in several lines.

There are several lines of text here

The example above uses the following LESS code:

    .progress-circle-text-size-example-1
    {
      .progressCircleColor(transparent, transparent, @orange, @orange);

      .text
      {
        .border-radius(50%);

        background-color: @white;
        border: 5px solid @green;
        font-weight: bold;
        margin: 4px;
      }

      &.progress-circle-large
      {
        .progressCircleSize(4, (5px + 4px), 3); // for the third parameter, the text in the element wraps to 3 lines

        .text
        {
          font-size: (@baseFontSize * 1.75);
        }
      }
    }

The CSS output would look like the following:

    .progress-circle-text-size-example-1 .progress-circle-bg {
      background-color: transparent;
      border-color: transparent;
    }
    .progress-circle-text-size-example-1 .bar {
      border-color: #f89406;
    }
    .progress-circle-text-size-example-1 .text {
      color: #f89406;
    }
    .progress-circle-text-size-example-1 .text {
      -webkit-border-radius: 50%;
      -moz-border-radius: 50%;
      border-radius: 50%;
      background-color: #ffffff;
      border: 5px solid #46a546;
      font-weight: bold;
      margin: 4px;
    }
    .progress-circle-text-size-example-1.progress-circle-large {
      width: 200px;
      height: 200px;
    }
    .progress-circle-text-size-example-1.progress-circle-large .progress-circle-bg,
    .progress-circle-text-size-example-1.progress-circle-large .bar {
      border-width: 20px;
    }
    .progress-circle-text-size-example-1.progress-circle-large .text {
      padding: 31px 0;
      font-size: 56px;
      line-height: 26.66666667px;
    }
    .progress-circle-text-size-example-1.progress-circle-large .text {
      font-size: 24.5px;
    }
